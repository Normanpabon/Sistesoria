"Instalación¨
1. Si se usa vscode abrir una nueva terminal, sino en cmd ir a la carpeta frontend en el pc y escribir el siguiente comando:
npm install

Las dependencias se instalan solas

Para probar el front end:

1. En cmd o una terminal de vscode, correr el siguiente comando en el directorio del frontend:

npm run dev

Esto habilitara el frontend en el siguiente servidor local:
"http://localhost:5173/"
